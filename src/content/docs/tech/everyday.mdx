### 初期化
- (PowerShell)usb ssdを接続
```txt
wsl --mount \\.\PHYSICALDRIVE1 --partition 1 --bare
```
- usbのデバイス名を確認
```txt
lsblk
```
- マウント
```txt
 sudo mount -t ext4 /dev/sde1 /mnt/usb
```
- NFSをsambaマウント
```txt
sudo mount -t cifs //192.168.1.1/ndisk /mnt/samba -o "username=nobe,password=Kisamic8,vers=2.0,uid=1000,gid=1000,file_mode=0644,dir_mode=0755"
```

### portforward
- (powershell) FW設定確認
```txt
Get-NetFirewallRule | Where-Object DisplayName -like "*WSL2*" | Get-NetFirewallPortFilter | Select-Object Protocol, LocalPort
```

- (powershell) FW設定(astro)
```txt
netsh advfirewall firewall add rule name="WSL2 astro" dir=in action=allow protocol=TCP localport=4321

```
- (powershell) FW設定(three.js)
```txt
netsh advfirewall firewall add rule name="WSL2 three.js" dir=in action=allow protocol=TCP localport=5173

```

- (powershell)port forward確認(されていることを確認するだけでいい何故なら、永続設定のはずなので）
```txt
netsh interface portproxy show all
```
